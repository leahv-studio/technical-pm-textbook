Common HTTP Status Codes

            

                

                    200
                    OK — The request succeeded and the server returned the requested data.
                

                

                    201
                    Created — A new resource was successfully created on the server.
                

                

                    400
                    Bad Request — The server couldn't understand the request due to invalid input.
                

                

                    401
                    Unauthorized — Authentication is required or the credentials are invalid.
                

                

                    404
                    Not Found — The requested resource doesn't exist at this endpoint.
                

                

                    500
                    Server Error — Something went wrong on the server side, not your fault.
